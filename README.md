# Useful PHP utils by Petr Knap

Useful PHP utilities by [Petr Knap].

## How to install

Add this code into your project `composer.json` file.

```json
{
    "repositories": [
        {
            "type": "git",
            "url": "https://github.com/petrknap/utils.git"
        }
    ],
    "require": {
        "petrknap/utils": "*"
    }
}
```

Run `composer install`.



[Petr Knap]:http://petrknap.cz/
